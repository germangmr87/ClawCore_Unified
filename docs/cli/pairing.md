# `clawcore pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
clawcore pairing list whatsapp
clawcore pairing approve whatsapp <code> --notify
```
